# ⚽ Прогноз — сервис футбольных прогнозов

Веб-сервис, который по предматчевым данным строит варианты прогнозируемых
результатов: исход 1X2, точные счёта, тоталы, «обе забьют», двойной шанс.

## Источники данных

| Сайт | Что берём | Как |
|------|-----------|-----|
| **stavka.tv** | коэффициенты 1X2, форма, статистика, прогнозы сообщества, текстовый разбор | публичный JSON API Nuxt (`/api/v2/matches/center`) |
| **arbworld.net** | прогрузы (dropping odds): движение линии и объёмы ставок | парсинг HTML-таблицы `/dropping-odds/football/1x2` |

## Как это работает

1. **Коэффициенты → вероятности.** Берём линию 1X2 со stavka.tv и переводим в
   вероятности с вычетом маржи букмекера.
2. **Модель Пуассона.** Подбираем ожидаемые голы (λ) хозяев и гостей так, чтобы
   модель воспроизводила эти вероятности 1X2.
3. **Прогрузы.** Сигналы с arbworld (куда падает линия / заходят деньги) смещают
   перевес в модели в сторону «умных денег».
4. **Варианты результатов.** Из итоговой модели считаем: 1X2, топ точных счетов,
   тотал 2.5, ОЗ, двойной шанс и текстовую рекомендацию с уровнем уверенности.

## Запуск

```bash
pip install -r requirements.txt
python app.py
# открыть http://127.0.0.1:5000
```

## Структура

```
app.py              веб-сервер (Flask): список матчей + карточка матча
prognoz/
  models.py         датаклассы (Match, Odds1x2, DropSignal, Prediction, ...)
  stavka.py         парсер stavka.tv (коэффициенты, форма, разбор)
  arbworld.py       парсер прогрузов arbworld
  matcher.py        связывает прогрузы с матчами по дате и именам команд
  predictor.py      движок: Пуассон + корректировка прогрузами
templates/          HTML-шаблоны
static/style.css    стили
```

## Ограничения

- Прогрузы привязываются к матчам по дате и английским названиям команд —
  лучше всего работает для сборных (ЧМ), для клубов возможны пропуски.
- Прогноз основан на рыночной линии и движении денег — это вероятностная
  оценка, **не гарантия результата и не инвестиционная рекомендация**.
